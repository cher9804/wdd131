BPO Colombia - Website Planning Document

Site Name

Name: BPO Colombia

Description: BPO Colombia is a modern business process outsourcing platform that connects US companies with highly qualified candidates from Colombia. The name reflects our commitment to quality, efficiency, and leveraging Colombia’s growing talent pool.

Logo: The website will feature a custom logo that blends modern design with subtle nods to Colombian culture.

Site Purpose

The primary purpose of BPO Colombia is to serve as a comprehensive hub that provides:

  • Connections between US companies and skilled professionals from Colombia.
  • Detailed information on outsourcing services and processes.
  • Resources for both companies and candidates regarding outsourcing benefits.
  • A user-friendly contact form for inquiries and service requests.

This platform aims to streamline the outsourcing process and enhance communication between businesses and talent.

Scenarios

To drive content development and ensure the site meets its audience’s needs, here are some key scenarios:

  • For Companies: "How can BPO Colombia help me efficiently source top talent from Colombia?"
  • For Candidates: "What opportunities does BPO Colombia offer for professionals seeking to work with US companies?"

Color Schema

The selected color scheme for BPO Colombia uses a modern pastel palette:

  • Primary Color: #a2d5f2 – used for headers, navigation bars, and call-to-action buttons.
  • Secondary Color: #f7a1b5 – applied for accents and highlights.
  • Accent Color: #ffefd5 – used for background elements and subtle design touches.

This color scheme provides a fresh, professional, and friendly look that resonates with the outsourcing industry.

Typography

The website will incorporate the following fonts to maintain clarity and style:

  • Heading Font: 'Roboto Slab', serif – for all main headings and titles to ensure a strong, modern appearance.
  • Body Font: 'Poppins', sans-serif – for all paragraph text and general content, ensuring readability.

This combination strikes the perfect balance between sophistication and readability.

Wireframe

The homepage layout for BPO Colombia has been designed for both mobile and desktop views:

Mobile View

  • A header with the logo and a vertical navigation menu.
  • A hero section with an engaging image and a key call-to-action.
  • A single-column layout that sequentially displays sections such as About, Services, Pricing, Impact, and Contact.

Desktop View

  • A horizontal navigation bar with the logo positioned on the left.
  • A full-width header section featuring a prominent hero image and call-to-action.
  • Multi-column layout for content sections to make full use of the screen space.

Wireframe sketches will be created using digital tools like Figma or Adobe XD, and they will be refined as the project evolves.

CSS

The supporting CSS will ensure the site is:

  • Responsive, using media queries to adjust for mobile and desktop layouts.
  • Consistent with the chosen color schema and typography.
  • Optimized for performance and accessibility, adhering to modern web standards.

This will help create a visually appealing and user-friendly experience across all devices.

Testing

The website will be thoroughly tested using:

  • HTML and CSS validators to ensure code quality and standards compliance.
  • Accessibility testing tools such as Lighthouse and WAVE to meet WCAG guidelines.
  • Performance evaluations and SEO audits to optimize the user experience.
  • Color contrast checks to ensure visual accessibility.

This comprehensive testing strategy will guarantee that BPO Colombia meets industry best practices.
